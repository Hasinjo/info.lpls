Chapitre 1 : Introduction à la base de données (BDD) avec Excel et sécurité informatique

1.1 Introduction à la base de données avec Excel

Description des concepts de base des systèmes d'information

Définition :

  • Base de données (BDD) : Une base de données est une collection organisée de données, généralement stockée et accessible électroniquement. Elle permet de stocker, manipuler et récupérer des informations de manière efficace.
  • Système d'information : Un système d'information est un ensemble de ressources (matérielles, logicielles, humaines) qui permet de collecter, stocker, traiter et diffuser des informations.

Exemple :

  • Une base de données peut être utilisée pour gérer les informations des élèves d'une école : noms, prénoms, âges, notes, etc.
  • Un système d'information scolaire peut inclure une base de données, des logiciels de gestion des notes, et des outils de communication avec les parents.

Exercice pratique :

  1. Ouvrez Excel et créez une nouvelle feuille de calcul.
  2. Créez une base de données simple avec les colonnes suivantes : Nom, Prénom, Âge, Note.
  3. Ajoutez 5 lignes de données fictives (exemple : Alice Dupont, 16 ans, 15/20).
  4. Utilisez la fonction de tri pour classer les élèves par ordre alphabétique de nom.
  5. Utilisez la fonction de filtre pour afficher uniquement les élèves ayant une note supérieure à 10.

Méthodes de conception de BDD

Définition :

  • Modélisation : La modélisation consiste à créer un modèle de données qui représente la structure et les relations entre les données. Un exemple courant est le diagramme entité-relation (ER).
  • Normalisation : La normalisation est un processus qui consiste à organiser les données pour éviter les redondances et les anomalies. Elle permet de garantir l'intégrité des données.

Exercice pratique :

  1. Créez un diagramme entité-relation simple pour une base de données de bibliothèque avec les entités suivantes : Livre, Auteur, Emprunteur.
  2. Identifiez les relations entre ces entités (exemple : un livre est écrit par un auteur, un livre est emprunté par un emprunteur).
  3. Créez une base de données Excel en respectant les principes de normalisation (exemple : une table pour les livres, une table pour les auteurs, et une table pour les emprunteurs).

1.2 Sécurité informatique

1.2.1 Situation problème 1 : Virus informatique

Définition :

Un virus informatique est un programme malveillant qui s'installe sur un ordinateur sans l'autorisation de l'utilisateur. Il peut endommager les fichiers, ralentir le système ou voler des informations.

Exemple :

Un utilisateur ouvre une pièce jointe dans un e-mail suspect. Le fichier contient un virus qui infecte l'ordinateur et crypte les fichiers, rendant impossible leur accès.

Exercice pratique :

  1. Discutez en classe des précautions à prendre pour éviter les virus (exemple : ne pas ouvrir les pièces jointes suspectes, utiliser un antivirus).
  2. Installez un antivirus sur votre ordinateur (si possible) et effectuez une analyse complète du système.

1.2.2 Situation problème 2 : Phishing

Définition :

Le phishing est une technique utilisée par les cybercriminels pour tromper les utilisateurs et voler leurs informations personnelles (mots de passe, numéros de carte bancaire).

Exemple :

Un utilisateur reçoit un e-mail qui semble provenir de sa banque, lui demandant de cliquer sur un lien pour mettre à jour ses informations. Le lien redirige vers un site frauduleux qui vole les informations saisies.

Exercice pratique :

  1. Identifiez les signes d'un e-mail de phishing (exemple : adresse e-mail suspecte, fautes d'orthographe, demande d'informations personnelles).
  2. Créez un faux e-mail de phishing (à des fins éducatives) et discutez en classe des éléments qui le rendent suspect.

1.2.3 Objectifs de la sécurité informatique

Définition :

Les objectifs de la sécurité informatique sont de protéger les données contre les accès non autorisés, d'assurer la confidentialité, l'intégrité et la disponibilité des informations.

Exemple :

  • Confidentialité : Seules les personnes autorisées peuvent accéder aux données.
  • Intégrité : Les données ne doivent pas être modifiées de manière non autorisée.
  • Disponibilité : Les données doivent être accessibles lorsque nécessaire.

Exercice pratique :

  1. Discutez en classe des mesures pour garantir la confidentialité, l'intégrité et la disponibilité des données (exemple : chiffrement des données, sauvegardes régulières).
  2. Créez un mot de passe fort et expliquez pourquoi il est sécurisé (exemple : utilisation de lettres, chiffres et symboles).

1.2.4 Concepts de cybersécurité et cybercriminalité

Définition :

  • Cybersécurité : La cybersécurité est la pratique qui consiste à protéger les systèmes informatiques contre les attaques et les accès non autorisés.
  • Cybercriminalité : La cybercriminalité regroupe toutes les activités criminelles commises via Internet (exemple : piratage, vol de données).

Exemple :

Une entreprise subit une attaque par ransomware (logiciel qui crypte les données et demande une rançon pour les débloquer).

Exercice pratique :

  1. Recherchez un cas réel de cybercriminalité (exemple : une attaque de ransomware célèbre) et présentez-le à la classe.
  2. Discutez des mesures de cybersécurité que l'entreprise aurait pu mettre en place pour éviter l'attaque.

1.2.5 Protection des données

Définition :

La protection des données consiste à mettre en place des mesures pour empêcher l'accès non autorisé, la modification ou la destruction des données.

Exemple :

Utiliser des mots de passe forts, installer un antivirus, éviter de cliquer sur des liens suspects.

Exercice pratique :

  1. Créez une liste de bonnes pratiques pour protéger vos données personnelles en ligne.
  2. Installez un gestionnaire de mots de passe (si possible) et générez un mot de passe fort pour un site web.

Chapitre 2 : Algorithmes et organigrammes, introduction à la programmation Web

2.1 Algorithmes et organigrammes

Définition d'un algorithme : Un algorithme est une suite d'instructions précises et ordonnées permettant de résoudre un problème ou d'effectuer une tâche.

Représentation avec un organigramme : Un organigramme est un diagramme visuel qui représente un algorithme à l'aide de symboles standardisés (exemple : début/fin, décision, action).

Exemple d'algorithme :

  1. Début
  2. Demander à l'utilisateur d'entrer un nombre
  3. Si le nombre est positif, afficher "Positif"
  4. Sinon, afficher "Négatif"
  5. Fin

Exercice pratique :

  1. Écrivez un algorithme pour déterminer si un nombre est pair ou impair.
  2. Représentez cet algorithme sous forme d'organigramme.

2.2 Introduction à la programmation Web

2.2.1 HTML : Structure générale d'un document HTML

Definition : HTML (HyperText Markup Language) est un langage de balisage utilisé pour créer des pages web. Il définit la structure et le contenu d'une page.

Structure générale d'un document HTML :

<!DOCTYPE html>
<html>
<head>
    <title>Titre de la page</title>
</head>
<body>
    <h1>Bienvenue sur ma page</h1>
    <p>Ceci est un paragraphe.</p>
</body>
</html>

Exercice pratique :

  1. Ouvrez un éditeur de texte (exemple : Notepad, Visual Studio Code).
  2. Créez un fichier HTML avec la structure de base.
  3. Ajoutez un titre, un en-tête et un paragraphe.
  4. Ouvrez le fichier dans un navigateur pour voir le résultat.

2.2.2 HTML : Structure sémantique basique

Définition : La structure sémantique en HTML permet de donner du sens aux éléments d'une page web (exemple : <header>, <main>, <footer>).

Exemple :

<header>
    <h1>Mon Site Web</h1>
</header>
<main>
    <p>Bienvenue sur mon site web.</p>
</main>
<footer>
    <p>Copyright @ 2023</p>
</footer>

Exercice pratique :

  1. Modifiez votre fichier HTML pour inclure une structure sémantique (<header>, <main>, <footer>).
  2. Ajoutez du contenu dans chaque section.

2.2.3 HTML : Tableau avec HTML

Définition : Les tableaux en HTML sont utilisés pour organiser des données en lignes et en colonnes.

Exemple :

<table>
    <tr>
        <th>Nom</th>
        <th>Âge</th>
    </tr>
    <tr>
        <td>Alice</td>
        <td>20</td>
    </tr>
</table>

Exercice pratique :

  1. Ajoutez un tableau à votre page HTML avec deux colonnes : "Nom" et "Âge".
  2. Remplissez le tableau avec des données fictives.

2.2.4 CSS : Sélecteurs et propriétés de base

Définition : CSS (Cascading Style Sheets) est un langage utilisé pour styliser les pages web (couleurs, polices, mise en page).

Sélecteurs :

  • Éléments : Appliquer un style à tous les éléments d'un type (exemple : h1 { color: blue; })
  • ID : Appliquer un style à un élément unique (exemple : #titre { font-size: 24px; })
  • Classes : Appliquer un style à plusieurs éléments (exemple : .important { font-weight: bold; })

Propriétés de base :

  • color : Couleur du texte
  • font-size : Taille de la police
  • margin : Marge autour d'un élément

Exercice pratique :

  1. Créez un fichier CSS et liez-le à votre fichier HTML.
  2. Ajoutez des styles pour les en-têtes (h1), les paragraphes (p) et les tableaux.

2.2.5 Mini Projet d'application : Site web statique

Projet :

Pour mettre en pratique les compétences acquises, voici un projet simple :

  1. Créez un site web statique avec plusieurs pages (exemple : Accueil, À propos, Contact).
  2. Utilisez HTML pour structurer le contenu et CSS pour le style.
  3. Ajoutez des éléments sémantiques (<header>, <main>, <footer>).
  4. Ajoutez un tableau pour afficher des informations (exemple : horaires d'ouverture).

Chapitre 3 : Premier pas avec le langage PYTHON et introduction à l'intelligence artificielle (IA)

3.1 Premier pas avec le langage PYTHON

3.1.1 Introduction à Python

Definition : Python est un langage de programmation interprété, facile à apprendre et à utiliser. Il est largement utilisé pour le développement web, l'analyse de données, l'automatisation, et bien plus encore.

Caractéristiques :

  • Syntaxe simple et lisible
  • Langage polyvalent (web, scientifique, IA, etc.)
  • Grande communauté et nombreuses bibliothèques disponibles

Installation : Python peut être installé sur Windows, macOS et Linux. Il est également possible d'utiliser des environnements en ligne comme Replit ou Google Colab.

3.1.2 Exemples de travaux pratiques

Exemples de code Python :

# Afficher un message
print("Bonjour, monde !")

# Utiliser des variables
nom = "Alice"
age = 16
print("Je m'appelle", nom, "et j'ai", age, "ans.")

# Boucle simple
for i in range(5):
    print("Itération numéro", i)

# Condition simple
nombre = 10
if nombre > 0:
    print("Le nombre est positif.")
else:
    print("Le nombre est négatif ou nul.")

Exercice pratique :

  1. Écrivez un programme Python qui demande à l'utilisateur son nom et son âge, puis affiche un message personnalisé.
  2. Écrivez un programme qui affiche les nombres pairs de 1 à 20 en utilisant une boucle for.

3.2 Introduction à l'intelligence artificielle (IA)

3.2.1 Définition de l'IA

Définition : L'intelligence artificielle (IA) est la capacité d'une machine à imiter l'intelligence humaine, en apprenant, en raisonnant et en s'adaptant à de nouvelles situations.

Applications courantes :

  • Reconnaissance faciale
  • Assistants virtuels (exemple : Siri, Alexa)
  • Voitures autonomes
  • Systèmes de recommandation (exemple : Netflix, YouTube)

3.2.2 Éthique pour l'éducation en lycée

Enjeux éthiques :

  • Respect de la vie privée : Les systèmes d'IA collectent souvent des données personnelles
  • Biais algorithmiques : Les IA peuvent reproduire ou amplifier les biais humains
  • Impact sur l'emploi : L'automatisation peut remplacer certains emplois

Discussion en classe :

  1. Quels sont les avantages et les inconvénients de l'IA dans la vie quotidienne ?
  2. Comment garantir que l'IA est utilisée de manière éthique et responsable ?

3.2.3 Exemples d'application de l'IA

Reconnaissance d'images :

Utilisation de l'IA pour identifier des objets dans des images (exemple : reconnaissance de visages, détection de maladies).

Traitement du langage naturel :

Utilisation de l'IA pour comprendre et générer du langage humain (exemple : chatbots, traduction automatique).

Exercice pratique :

  1. Recherchez un exemple d'application de l'IA dans un domaine qui vous intéresse (exemple : santé, éducation, divertissement).
  2. Présentez cet exemple à la classe en expliquant comment l'IA est utilisée et quels sont ses impacts.

3.2.4 Projet pratique : Créer un programme Python simple

Projet :

Pour mettre en pratique les compétences acquises, voici un projet simple :

  1. Créez un programme Python qui demande à l'utilisateur de deviner un nombre entre 1 et 100.
  2. Le programme doit indiquer si le nombre deviné est trop grand, trop petit, ou correct.
  3. Ajoutez un compteur pour afficher le nombre de tentatives nécessaires pour trouver le bon nombre.
  4. Testez votre programme et améliorez-le si nécessaire.
