1. Programmation avancée avec Scratch

Ce chapitre approfondit les concepts de programmation avec Scratch : coordonnées, boucles, variables, structures conditionnelles, dessins, sons, clavier, et blocs personnalisés. Ces notions sont indispensables pour créer des programmes interactifs et résoudre des problèmes mathématiques ou ludiques.

1.1 Notions de coordonnées

Définition : Les coordonnées sont des paires de nombres (x, y) qui définissent la position d’un point dans un plan cartésien. En Scratch, l’axe x est horizontal, l’axe y est vertical.

Exemple : La position (100, 50) place le sprite à 100 sur l’axe x et 50 sur l’axe y.

Exercice pratique

Créez un programme où un sprite dessine un carré en allant aux coordonnées (0,0), (100,0), (100,100), (0,100).

1.2 Création des instructions

Définition : Les instructions sont des commandes qui contrôlent les actions des sprites. Elles se combinent pour former des séquences logiques.

Bloc Action
avancer de 10 pas Déplace le sprite de 10 pas.
tourner de 90 degrés Fait pivoter le sprite de 90°.
aller à x:100 y:50 Déplace le sprite à la position (100,50).

Exercice pratique

Créez un programme où un sprite suit un triangle équilatéral avec des instructions de mouvement et de rotation.

1.3 Les boucles

Définition : Une boucle permet de répéter une série d’instructions plusieurs fois. En Scratch, on utilise le bloc répéter. 

répéter (24) fois
    avancer de 10 pas
    tourner de 15 degrés
fin

Exercice pratique

  • Dessiner un hexagone régulier avec une boucle
  • Dessiner une étoile à 5 branches
  • Dessiner une spirale carrée

1.4 Les mouvements

Définition : Les mouvements permettent de déplacer un sprite dans différentes directions (avancer, tourner, aller à…).

Exercice pratique

Créez un programme où un sprite suit une trajectoire en spirale ou en parabole.

1.5 Le stylo et les dessins

Définition : Le stylo permet de dessiner des lignes et des formes sur la scène en suivant les mouvements du sprite.

Exercice pratique

Créez un programme où un sprite dessine une étoile à 5 branches avec le stylo.

1.6 Le clavier

Définition : Le clavier permet d’interagir avec le programme en utilisant des touches pour déclencher des actions.

Exercice pratique

Créez un programme où un sprite se déplace selon les touches fléchées et trace une figure géométrique.

1.7 Le son et effets sonores

Définition : Les sons rendent le programme plus interactif. On peut jouer un son selon des événements.

Exercice pratique

Créez un programme où un sprite joue un son différent selon sa position sur l’axe x.

1.8 Les variables

Définition : Les variables stockent des informations (nombres, texte) que le programme peut utiliser et modifier.

Exercice pratique

Créez un programme où une variable affiche la distance parcourue par un sprite.

1.9 Si ... alors ... sinon

Définition : Les structures conditionnelles permettent d’exécuter des actions selon des conditions.

Exercice pratique

Créez un programme où un sprite change de direction s’il atteint un bord de l’écran.

1.10 Opérations algébriques

Définition : Les opérations algébriques (addition, soustraction, multiplication, division) manipulent des nombres dans le programme.

Exercice pratique

Créez un programme où un sprite suit une trajectoire définie par une équation mathématique (ex : y = x²).

1.11 Blocs

Définition : Les blocs regroupent des instructions pour créer des fonctions réutilisables.

Exercice pratique

Créez un bloc "dessiner un polygone régulier" qui prend en entrée le nombre de côtés.

2. Introduction à l’algorithme

Ce chapitre introduit la notion d’algorithme, les variables, instructions, opérateurs, et organigrammes. Des exercices pratiques et des applications Scratch permettent de consolider ces bases pour résoudre des problèmes logiques et mathématiques.

2.1 Structure générale d’un algorithme

Définition : Un algorithme est une suite d’instructions logiques pour résoudre un problème. Il peut être représenté sous forme de texte, de code ou d’organigramme.

Exemple : Algorithme pour calculer la somme de deux nombres :

  1. Demander à l’utilisateur de saisir le premier nombre.
  2. Demander à l’utilisateur de saisir le deuxième nombre.
  3. Ajouter les deux nombres.
  4. Afficher le résultat.

Exercice pratique

Écrivez un algorithme pour calculer la moyenne de trois nombres.

2.2 Variables et constantes

Définition : Les variables stockent des données modifiables ; les constantes stockent des valeurs fixes.

Élément Description
Variable Stocke des données modifiables (ex : âge).
Constante Stocke des données fixes (ex : π).

Exercice pratique

Créez un algorithme qui utilise une variable pour le rayon d’un cercle et une constante pour π, puis calcule la circonférence.

2.3 Instructions

Définition : Les instructions décrivent les actions à exécuter dans un algorithme.

Exercice pratique

Écrivez un algorithme qui demande le nom de l’utilisateur et affiche un message de bienvenue.

2.4 Opérateurs

Définition : Les opérateurs manipulent des valeurs (addition, soustraction, multiplication, division).

Opérateur Description
+ Addition
− Soustraction
× Multiplication
÷ Division

Exercice pratique

Écrivez un algorithme qui utilise des opérateurs pour calculer l’aire d’un rectangle.

2.5 Organigramme

Définition : Un organigramme est une représentation visuelle d’un algorithme avec des symboles (rectangles, losanges, flèches…).

Exercice pratique

Dessinez un organigramme pour déterminer si un nombre est positif, négatif ou nul.

2.6 Application avec Scratch et approche mathématique

Définition : Les algorithmes peuvent être appliqués dans Scratch pour résoudre des problèmes mathématiques (aire, périmètre…).

Exercice pratique

Créez un programme Scratch pour calculer le volume d’une sphère avec la formule V = (4/3) × π × r³.

3. Introduction à l’intelligence artificielle

Ce chapitre aborde les bases de l’intelligence artificielle : définition, éthique, prompts efficaces, mini-projets. L’objectif est de comprendre les enjeux, les applications et les limites de l’IA aujourd’hui.

3.1 Définition de l’intelligence artificielle

Définition : L’IA est la capacité d’une machine à imiter l’intelligence humaine (apprendre, raisonner, s’adapter).

Exemple : Siri, Alexa : assistants virtuels utilisant l’IA pour comprendre et répondre aux questions.

Exercice pratique

Discutez en classe des exemples d’IA que vous utilisez au quotidien.

3.2 Éthique de l’intelligence artificielle

Définition : L’éthique de l’IA traite des questions morales : protection des données, biais, impact sur l’emploi.

Problème éthique Description
Protection des données Comment les données personnelles sont-elles utilisées ?
Biais algorithmiques Les algorithmes peuvent-ils être injustes ou discriminatoires ?
Impact sur l’emploi L’IA remplace-t-elle des emplois humains ?

Exercice pratique

Débat : avantages et inconvénients de l’IA dans la société.

3.3 Prompt efficace

Définition : Un prompt est une instruction donnée à une IA pour obtenir une réponse précise. Un prompt efficace est clair, précis et structuré.

Exemple : "Explique-moi les causes de la Révolution française en 1789" est plus efficace que "Parle-moi de l’histoire".

Exercice pratique

Écrivez trois prompts efficaces pour demander à une IA de résoudre des problèmes mathématiques.

3.4 Mini-projets

Définition : Les mini-projets permettent de mettre en pratique les concepts d’IA à travers des applications concrètes.

Projet Description
Reconnaissance de formes Utiliser des règles simples pour identifier des formes géométriques.
Chatbot simple Créer un chatbot qui répond à des questions basiques.
Classification d’images Classer des images selon leur contenu.

Exercice pratique

Créez un programme Scratch où un sprite utilise des règles simples pour résoudre des problèmes mathématiques.
