Chapitre 1: Les fondements historiques et sociaux de la science informatique

1.1 Définitions importantes

  • Information : Donnée interprétée et utilisée pour prendre des décisions.
  • Donnée : Élément brut qui peut être traité pour produire de l'information.
  • Traitement : Processus de transformation des données en information.
  • Ordinateur : Machine électronique capable de traiter des données.
  • Informatique : Science du traitement automatique de l'information.

1.2 Historique des machines

Les premières machines

  • Pascaline (1642) : La première machine à calculer mécanique, inventée par Blaise Pascal.
  • Machine analytique de Charles Babbage (1837) : Une machine programmable considérée comme l'ancêtre des ordinateurs modernes.

Les ordinateurs modernes

  • ENIAC (1945) : Le premier ordinateur électronique programmable, utilisé pour des calculs militaires.
  • Premiers micro-ordinateurs (années 1970) : L'arrivée des ordinateurs personnels comme l'Apple II et le Commodore 64.

Évolution des technologies

Des ordinateurs centraux aux smartphones : Montrer comment les ordinateurs sont devenus plus petits, plus puissants et plus accessibles.

1.3 Champs d'application des ordinateurs

  • Calcul scientifique : Utilisation des ordinateurs pour des simulations et des modélisations complexes. Exemple : Prévisions météorologiques.
  • Contrôle des processus : Automatisation des processus industriels. Exemple : Robots dans les usines.
  • Robotique : Utilisation des robots dans divers domaines. Exemple : Robots médicaux, robots domestiques.
  • Jeux vidéo : Conception et développement de jeux vidéo. Exemple : Jeux comme Minecraft ou Fortnite.
  • Bureautique : Utilisation des ordinateurs pour le traitement de texte, les tableurs et les présentations. Exemple : Microsoft Word, Excel.
  • Communication : Utilisation des ordinateurs pour communiquer. Exemple : E-mails, réseaux sociaux.
  • Cinéma : Utilisation des ordinateurs pour les effets spéciaux et l'animation. Exemple : Films comme Avatar.
  • Graphisme : Conception graphique et infographie. Exemple : Logiciels comme Photoshop.
  • Informatique pédagogique : Utilisation des ordinateurs pour l'éducation. Exemple : Logiciels éducatifs, cours en ligne.

1.4 Organes de base d'un ordinateur

Organes essentiels

  • Unité centrale : Le cerveau de l'ordinateur, composé du processeur, de la mémoire et du disque dur.
  • Écran : Affiche les informations traitées par l'ordinateur.
  • Clavier et souris : Permettent d'interagir avec l'ordinateur.

Périphériques

  • Imprimante : Permet d'imprimer des documents.
  • Scanner : Permet de numériser des documents papier.
  • Modem : Permet de se connecter à Internet.
  • Clé USB : Permet de stocker et transporter des données.

1.5 Exercice pratique

Pour mettre en pratique les connaissances acquises, voici un exercice simple :

  1. Recherche historique : Recherchez une machine historique (exemple : Pascaline, ENIAC) et présentez son fonctionnement en quelques phrases.
  2. Identification des composants : Identifiez les organes essentiels (unité centrale, écran, clavier, souris) et les périphériques (imprimante, scanner, modem) sur une image d'ordinateur.
  3. Discussion sur les applications : Choisissez un champ d'application des ordinateurs (exemple : jeux vidéo, cinéma) et expliquez comment les ordinateurs sont utilisés dans ce domaine.

Chapitre 2: Introduction au traitement de texte

2.1 Microsoft Office Word

Présentation

Microsoft Word est un logiciel de traitement de texte largement utilisé pour créer des documents professionnels, scolaires ou personnels.

Fonctionnalités de base

  • Saisie de texte : Apprendre à taper du texte dans un document.
  • Mise en forme : Modifier la police, la taille, la couleur et les styles (gras, italique, souligné).
  • Insertion d'éléments : Ajouter des images, des tableaux et des graphiques.
  • Enregistrement et impression : Sauvegarder un document et l'imprimer.

Exercice pratique

  1. Ouvrez Microsoft Word.
  2. Saisissez un texte court (exemple : "Bonjour, je m'appelle [votre nom].").
  3. Modifiez la police en Arial, la taille en 14, et appliquez le style gras.
  4. Insérez une image à partir de votre ordinateur.
  5. Enregistrez le document sous le nom "MonPremierDocument.docx".

2.2 LibreOffice Writer

Présentation

LibreOffice Writer est une alternative gratuite et open-source à Microsoft Word. Il offre des fonctionnalités similaires pour le traitement de texte.

Fonctionnalités de base

  • Saisie de texte : Taper du texte dans un document.
  • Mise en forme : Changer la police, la taille, la couleur et les styles.
  • Insertion d'éléments : Ajouter des images, des tableaux et des graphiques.
  • Enregistrement : Sauvegarder un document au format ODT ou PDF.

Exercice pratique

  1. Ouvrez LibreOffice Writer.
  2. Saisissez un texte court (exemple : "Ceci est un document créé avec LibreOffice.").
  3. Modifiez la police en Times New Roman, la taille en 12, et appliquez le style italique.
  4. Insérez un tableau simple avec 2 colonnes et 3 lignes.
  5. Enregistrez le document sous le nom "MonDocumentLibreOffice.odt".

2.3 Comparaison entre Microsoft Word et LibreOffice Writer

  1. Microsoft Word :
    • Logiciel payant (inclus dans Microsoft Office).
    • Interface moderne et intuitive.
    • Compatibilité élevée avec les formats de fichiers courants (DOCX, PDF).
  2. LibreOffice Writer :
    • Logiciel gratuit et open-source.
    • Interface simple mais efficace.
    • Formats de fichiers natifs (ODT), mais compatible avec DOCX et PDF.

2.4 Mini-projet : Créer un document de présentation

Pour mettre en pratique les compétences acquises, voici un projet simple :

  1. Choisissez un sujet (exemple : votre animal préféré, un pays, un sport).
  2. Créez un document de présentation avec Microsoft Word ou LibreOffice Writer.
  3. Incluez les éléments suivants :
    • Un titre mis en forme (gras, taille 16, centré).
    • Un paragraphe de texte décrivant le sujet.
    • Une image illustrant le sujet.
    • Un tableau avec des informations complémentaires (exemple : dates, chiffres).
  4. Enregistrez le document et imprimez-le si possible.

Chapitre 3: Initiation à l'Internet

3.1 Définition de l'internet

C'est un réseau mondial qui relie des millions d'ordinateurs et d'appareils à travers le monde. Il permet d'accéder à des informations, de communiquer et de partager des données.

Usages courants d'Internet

  • Recherche d'information : Trouver des réponses à des questions, des articles, des vidéos, etc.
  • Communication : Envoyer des e-mails, discuter en ligne, utiliser les réseaux sociaux.
  • Divertissement : Regarder des vidéos, écouter de la musique, jouer à des jeux en ligne.
  • Éducation : Suivre des cours en ligne, accéder à des ressources pédagogiques.

3.2 Comment accéder à Internet ?

Les moyens de connexion

  • Wi-Fi : Connexion sans fil à un réseau local.
  • Câble Ethernet : Connexion filaire pour une connexion plus stable.
  • Données mobiles : Connexion via un réseau de téléphonie mobile.

Les appareils connectés

  • Ordinateurs,
  • tablettes,
  • smartphones,
  • objets connectés.

3.3 Les navigateurs web

Qu'est-ce qu'un navigateur web ?

Un navigateur web est un logiciel qui permet d'accéder à Internet et de consulter des sites web.

Exemples de navigateurs

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Fonctionnalités de base

  • Onglets : Permet d'ouvrir plusieurs pages web dans une même fenêtre.
  • Favoris : Enregistrer des sites web pour y accéder rapidement.
  • Historique : Consulter les pages web visitées précédemment.

3.4 Recherche sur Internet

Qu'est-ce qu'un moteur de recherche ?

Un moteur de recherche est un outil qui permet de trouver des informations sur Internet en tapant des mots-clés.

Exemples de moteurs de recherche

  • Google
  • Bing
  • Yahoo

Techniques de recherche efficaces

  • Utiliser des mots-clés précis.
  • Utiliser les guillemets pour rechercher une expression exacte.
  • Utiliser les opérateurs de recherche (exemple : site: pour limiter la recherche à un site spécifique).

Évaluation des sources

  • Vérifier la fiabilité des sources (sites officiels, articles de qualité).
  • Éviter les informations non vérifiées ou trompeuses.

3.5 Exercice pratique : Recherche d'information sur Internet

Pour mettre en pratique les compétences acquises, voici un exercice simple :

  1. Choisissez un sujet qui vous intéresse (exemple : les animaux, les pays, les inventions).
  2. Utilisez un moteur de recherche pour trouver des informations fiables sur ce sujet.
  3. Rassemblez les informations dans un document texte.
  4. Présentez vos résultats à la classe en utilisant les techniques de communication apprises.

Chapitre 4: Premiers pas avec la programmation SCRATCH

4.1 Définition : SCRATCH

Le SCRATCH est un langage de programmation visuel conçu pour les débutants. Il permet de créer des animations, des jeux et des histoires interactives en assemblant des blocs de code.

4.2 Interface de SCRATCH

  • Scène : L'espace où les animations et les dessins sont affichés.
  • Lutins : Les personnages ou objets qui interagissent sur la scène.
  • Blocs de programmation : Les instructions que l'on assemble pour créer des programmes.

Objectif

Apprendre à créer des programmes simples en utilisant des blocs de code.

4.3 Les bases de la programmation avec SCRATCH

4.3.1 Mouvement

  • Utiliser les blocs de mouvement pour déplacer un lutin sur la scène.
  • Exemple : Faire avancer un lutin de 10 pas, puis le faire tourner de 90 degrés.

4.3.2 Apparence

  • Changer l'apparence d'un lutin (costume, taille, effet visuel).
  • Exemple : Faire dire "Bonjour!" à un lutin pendant 2 secondes.

4.3.3 Événements

  • Utiliser les blocs d'événements pour déclencher des actions.
  • Exemple : Démarrer un programme lorsque l'utilisateur clique sur le drapeau vert.

4.4 Créer un premier programme

4.4.1 Étapes pour créer un programme simple

  1. Ouvrez SCRATCH et créez un nouveau projet.
  2. Sélectionnez un lutin (exemple : le chat).
  3. Utilisez les blocs de mouvement pour faire bouger le lutin.
  4. Ajoutez un bloc d'apparence pour faire dire "Bonjour!" au lutin.
  5. Utilisez un bloc d'événement pour démarrer le programme lorsque l'utilisateur clique sur le drapeau vert.

4.4.2 Exemple de programme

  • Faire avancer le chat de 50 pas.
  • Faire dire "Bonjour!" pendant 2 secondes.
  • Faire tourner le chat de 90 degrés.

4.5 Faire des dessins avec SCRATCH

  1. Utilisation du stylo : SCRATCH permet de dessiner sur la scène en utilisant l'extension "Stylo".
  2. Blocs de base du stylo :
    • stylo en bas : Active le stylo pour dessiner.
    • stylo en haut : Désactive le stylo.
    • effacer tout : Efface tous les dessins de la scène.
    • changer la couleur du stylo : Modifie la couleur du tracé.
  3. Dessiner des formes géométriques :
    • Carré :
      1. Activer le stylo avec stylo en bas.
      2. Faire avancer le lutin de 100 pas.
      3. Faire tourner le lutin de 90 degrés.
      4. Répéter les étapes 2 et 3 trois fois pour compléter le carré.
    • Triangle :
      1. Activer le stylo avec stylo en bas.
      2. Faire avancer le lutin de 100 pas.
      3. Faire tourner le lutin de 120 degrés.
      4. Répéter les étapes 2 et 3 deux fois pour compléter le triangle.
    • Rectangle :
      1. Activer le stylo avec stylo en bas.
      2. Faire avancer le lutin de 150 pas.
      3. Faire tourner le lutin de 90 degrés.
      4. Faire avancer le lutin de 100 pas.
      5. Faire tourner le lutin de 90 degrés.
      6. Répéter les étapes 2 à 5 une fois pour compléter le rectangle.

4.6 Exercice pratique : Dessiner une maison avec SCRATCH

Pour mettre en pratique les compétences acquises, voici un projet simple :

  1. Créez un nouveau projet dans SCRATCH.
  2. Ajoutez un lutin (exemple : un crayon ou un personnage).
  3. Utilisez les blocs de mouvement et le stylo pour dessiner une maison :
    • Dessinez un carré pour la base de la maison.
    • Dessinez un triangle pour le toit.
    • Ajoutez une porte et une fenêtre en utilisant des lignes supplémentaires.
  4. Testez votre programme et ajustez les paramètres si nécessaire.
