Exercice HTML/CSS Facile

Consignes :

Créez une page personnelle avec :

  1. Un en-tête avec votre prénom
  2. Une section contenant vos loisirs
  3. Un pied de page avec le copyright

Correction HTML :

<!DOCTYPE html> <!-- 1. Déclaration du type de document --> <html lang="fr"> <!-- 2. Balise racine avec langue --> <head> <!-- 3. En-tête invisible --> <meta charset="UTF-8"> <!-- 4. Encodage des caractères --> <title>Ma Page</title> <!-- 5. Titre de l'onglet --> <link rel="stylesheet" href="style.css"> <!-- 6. Lien vers le CSS --> </head> <body> <!-- 7. Début du contenu visible --> <header> <!-- 8. En-tête visible --> <h1>Sophie</h1> <!-- 9. Titre principal --> </header> <main> <!-- 10. Contenu principal --> <section> <!-- 11. Section thématique --> <h2>Mes Loisirs</h2> <!-- 12. Sous-titre --> <ul> <!-- 13. Liste non ordonnée --> <li>Dessin</li> <!-- 14. Élément de liste --> <li>Musique</li> <li>Sport</li> </ul> </section> </main> <footer> <!-- 15. Pied de page --> <p>© 2023 - Sophie</p> <!-- 16. Copyright --> </footer> </body> </html>

Correction CSS :

/* 1. Style du corps de la page */ body { background-color: #f0f8ff; /* Fond bleu très clair */ font-family: Arial; /* Police simple */ margin: 0; /* Supprime les marges par défaut */ padding: 20px; /* Espace autour du contenu */ } /* 2. Style de l'en-tête */ header { text-align: center; /* Centrage du texte */ padding: 20px; /* Espacement interne */ } /* 3. Style du titre principal */ h1 { color: #0066cc; /* Couleur bleue */ font-size: 2em; /* Taille du texte */ } /* 4. Style de la section */ section { background-color: white; /* Fond blanc */ padding: 15px; /* Espacement interne */ border-radius: 8px; /* Coins arrondis */ margin: 15px 0; /* Marge externe */ } /* 5. Style de la liste */ ul { font-family: cursive; /* Police cursive */ color: #333; /* Couleur de texte */ } /* 6. Style du pied de page */ footer { text-align: center; /* Centrage du texte */ background-color: #eee; /* Fond gris clair */ padding: 10px; /* Espacement interne */ font-size: 0.9em; /* Taille de texte réduite */ }

Explications :

Structure HTML :

Balises sémantiques : header, main, section, footer aident à structurer la page.

Hiérarchie : h1 pour le titre principal, h2 pour les sous-titres.

Liste : ul/li pour créer une liste à puces non ordonnée.

Style CSS :

Sélecteurs : On cible les balises (body, header, h1...).

Couleurs : On utilise des codes hexadécimaux (#0066cc).

Espacement : padding pour l'intérieur, margin pour l'extérieur.

Alignement : text-align: center pour centrer.
