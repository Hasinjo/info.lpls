Exercice HTML/CSS pour Débutants

Consignes de l'exercice

Créez une page web personnelle contenant :

  1. Un en-tête avec votre prénom
  2. Une section avec vos centres d'intérêt
  3. Un pied de page avec mention de copyright

Avec ces styles CSS :

  • Fond de page bleu clair
  • Titre principal centré et bleu foncé
  • Section avec fond blanc et ombre portée
  • Liste en police cursive

Correction Complète

Structure HTML

<!DOCTYPE html> <html lang="fr"> <head> <meta charset="UTF-8"> <title>Ma Page Personnelle</title> <link rel="stylesheet" href="style.css"> </head> <body> <header> <h1>Mon Prénom</h1> </header> <main> <section> <h2>Mes Centres d'Intérêt</h2> <ul> <li>Musique</li> <li>Sport</li> <li>Voyages</li> </ul> </section> </main> <footer> <p>&copy; 2023 - Mon Prénom</p> </footer> </body> </html>

Explications HTML

1 DOCTYPE : Définit le type de document HTML5

2 Balises sémantiques : header, main, section, footer structurent la page

3 Hiérarchie : h1 pour le titre principal, h2 pour le sous-titre

4 Liste : ul/li crée une liste à puces non ordonnée

Feuille de Style CSS

/* Style général de la page */ body { background-color: #f0f8ff; /* Fond bleu très clair */ font-family: Arial, sans-serif; margin: 0; padding: 20px; } /* En-tête de page */ header { text-align: center; /* Texte centré */ padding: 30px 0; /* Espacement */ } /* Titre principal */ h1 { color: #0066cc; /* Couleur bleue */ font-size: 2.2em; /* Taille du texte */ margin-bottom: 0; /* Supprime la marge */ } /* Section principale */ section { background-color: white; /* Fond blanc */ padding: 20px; /* Espacement interne */ border-radius: 8px; /* Coins arrondis */ box-shadow: 0 2px 5px rgba(0,0,0,0.1); /* Ombre légère */ max-width: 600px; /* Largeur maximale */ margin: 20px auto; /* Centrage */ } /* Style de la liste */ ul { font-family: cursive; /* Police cursive */ color: #333; /* Couleur de texte */ line-height: 1.8; /* Hauteur de ligne */ padding-left: 20px; /* Alignement des puces */ } /* Pied de page */ footer { text-align: center; /* Centrage du texte */ background-color: #e0e0e0; /* Fond gris clair */ padding: 15px; /* Espacement */ font-size: 0.9em; /* Taille réduite */ margin-top: 30px; /* Espace au-dessus */ }

Explications CSS

1 Sélecteurs : On cible les éléments HTML à styliser

2 Couleurs : Utilisation de codes hexadécimaux (#0066cc)

3 Espacement : padding pour l'intérieur, margin pour l'extérieur

4 Typographie : font-family pour changer la police

Astuce : Les commentaires CSS (/* ... */) aident à organiser son code.
