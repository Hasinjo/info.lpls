Chapitre 1 : Programmation avec Python

Introduction

Ce chapitre introduit les concepts de base de la programmation en Python, en mettant l'accent sur les variables, les conditions, les fonctions, les boucles, et les structures de données.

Historique de Python

Python a été créé par Guido van Rossum à la fin des années 1980. Le langage a été conçu pour être facile à lire et à écrire, avec une syntaxe claire et concise. Voici quelques étapes clés de son histoire :

  • 1989 : Début du développement de Python.
  • 1991 : Première version publique de Python (0.9.0).
  • 2000 : Sortie de Python 2.0, introduisant de nouvelles fonctionnalités comme les list comprehensions.
  • 2008 : Sortie de Python 3.0, une version majeure qui n'est pas rétrocompatible avec Python 2.
  • Aujourd'hui : Python est l'un des langages de programmation les plus populaires, utilisé dans des domaines variés comme le développement web, l'analyse de données, l'intelligence artificielle, et plus encore.

1.1 Définition

La programmation consiste à écrire des instructions (code) pour qu'un ordinateur exécute des tâches spécifiques.

Exemple :

print("Bonjour le monde")

Méthode

  • Comprendre le problème.
  • Écrire du code simple.
  • Tester et déboguer.
  • Utiliser un environnement de développement.

Travaux pratiques

Écrire un programme qui demande à l'utilisateur son nom et affiche un message de bienvenue.

1.2 Variables et expressions

Définition

Une variable est un conteneur pour stocker des données. Une expression est une combinaison de variables, opérateurs et valeurs.

Exemple :

x = 10

y = 5

somme = x + y

print(somme) # Affiche 15

Méthode

  • Déclarer des variables.
  • Utiliser des opérateurs arithmétiques.
  • Comprendre les types de données.
  • Écrire des expressions.

Travaux pratiques

Créer un programme qui calcule l'aire d'un rectangle.

1.3 Exécution conditionnelle

Définition

L'exécution conditionnelle permet d'exécuter du code en fonction de certaines conditions.

Exemple :

age = 18

if age >= 18:

print("Vous êtes majeur.")

else:

print("Vous êtes mineur.")

Méthode

  • Comprendre les conditions.
  • Utiliser if, else, elif.
  • Imbriquer des conditions.
  • Tester toutes les possibilités.

Travaux pratiques

Écrire un programme qui vérifie si un nombre est positif, négatif ou nul.

1.4 Fonctions

Définition

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique.

Exemple :

def dire_bonjour(nom):

print(f"Bonjour, {nom}!")

dire_bonjour("Alice")

Méthode

  • Définir une fonction avec def.
  • Utiliser des paramètres.
  • Retourner des valeurs avec return.
  • Documenter les fonctions.

Travaux pratiques

Créer une fonction qui calcule la factorielle d'un nombre.

1.5 Itérations

Définition

Les itérations permettent de répéter des actions à l'aide de boucles.

Exemple :

for i in range(5):

print(i) # Affiche 0, 1, 2, 3, 4

Méthode

  • Utiliser for pour des boucles définies.
  • Utiliser while pour des boucles indéfinies.
  • Éviter les boucles infinies.
  • Utiliser break et continue.

Travaux pratiques

Écrire un programme qui affiche les 10 premiers nombres pairs.

1.6 Listes, Dictionnaires, Ensembles

Définition

  • Liste : Une collection ordonnée d'éléments.
  • Dictionnaire : Une collection de paires clé-valeur.
  • Ensemble : Une collection non ordonnée d'éléments uniques.

Exemple :

liste = [1, 2, 3]

dictionnaire = {"nom": "Alice", "âge": 25}

ensemble = {1, 2, 3}

Méthode

  • Manipuler les listes.
  • Utiliser les dictionnaires.
  • Travailler avec les ensembles.
  • Comprendre les méthodes courantes.

Travaux pratiques

Créer un programme qui stocke les notes des élèves dans une liste et calcule la moyenne.

Chapitre 3 : Introduction à la cybersécurité

Introduction

Ce chapitre introduit les concepts de base de la cybersécurité, en mettant l'accent sur la protection des systèmes, des réseaux et des données contre les attaques numériques.

Historique de la cybersécurité

La cybersécurité a évolué avec l'essor de l'informatique et d'Internet. Voici quelques étapes clés :

  • Années 1970 : Premiers virus informatiques comme "Creeper".
  • Années 1980 : Création des premiers antivirus.
  • Années 1990 : Augmentation des cyberattaques (vers, chevaux de Troie).
  • Années 2000 : Attaques sophistiquées (ransomwares, DDoS).
  • Aujourd'hui : Enjeu majeur contre le phishing, attaques zero-day, etc.

3.1 Définitions

3.1.1 Qu'est-ce que la cybersécurité ?

Pratique de protéger systèmes, réseaux et programmes contre les attaques numériques visant à accéder, modifier ou détruire des informations sensibles.

3.1.2 Objectifs

  • Confidentialité : Protéger l'accès aux informations
  • Intégrité : Empêcher la modification non autorisée
  • Disponibilité : Garantir l'accès aux systèmes

3.2 Notions de base et attaques populaires

3.2.1 Notions de base

  • Virus : Programme malveillant qui infecte d'autres programmes
  • Ver : Se propage sans intervention humaine
  • Cheval de Troie : Logiciel malveillant déguisé
  • Phishing : Technique pour voler des informations

3.2.2 Attaques populaires

  • Ransomware : Chiffre les données et demande une rançon
  • DDoS : Submerge un service de trafic pour le rendre indisponible
  • Attaque zero-day : Exploite une vulnérabilité inconnue
  • Ingénierie sociale : Manipulation psychologique

3.3 Exemples pratiques

3.3.1 Protéger son ordinateur

  • Installer un antivirus à jour
  • Utiliser un pare-feu
  • Mettre à jour les logiciels

3.3.2 Protéger ses comptes en ligne

  • Mots de passe forts et uniques
  • Authentification à deux facteurs (2FA)
  • Éviter les liens suspects

3.3.3 Travaux pratiques

  • Créer un mot de passe fort et expliquer sa sécurité
  • Configurer les paramètres de sécurité de base
  • Analyser un email de phishing

4.1 Structure générale d'un algorithme

Définition

Une séquence d'étapes bien définies pour résoudre un problème.

Composants

  • Entrée : Données fournies
  • Traitement : Étapes de calcul
  • Sortie : Résultat produit

Exemple : Algorithme de somme

1. Lire le premier nombre (A)

2. Lire le deuxième nombre (B)

3. Calculer S = A + B

4. Afficher S

4.2 Variables et constantes

  • Constante : Valeur fixe (ex: PI = 3.14)

4.3 Instructions

  • Conditionnelle : Si x > 10 alors...
  • Itération : Pour i de 1 à 10 faire...
