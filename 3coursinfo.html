Chapitre 1 : Programmation avancée avec Scratch

Dans ce chapitre, nous allons explorer les concepts avancés de Scratch pour créer des programmes interactifs et résoudre des problèmes mathématiques. Vous apprendrez à utiliser les coordonnées, les boucles, les variables et bien plus encore !

1.1 Notions de coordonnées

Définition : Les coordonnées sont des paires de nombres (x, y) qui définissent la position d'un point dans un plan cartésien.

  • Axe x : position horizontale
  • Axe y : position verticale

Exemple : La position (100, 50) place le sprite à 100 unités à droite et 50 unités vers le haut.

Figure 1.1 : Représentation des coordonnées dans Scratch

Exercice pratique

Créez un programme où un sprite se déplace pour dessiner un carré en utilisant les coordonnées :

  • (0, 0)
  • (100, 0)
  • (100, 100)
  • (0, 100)

1.2 Création des instructions

Définition : Les instructions sont des commandes qui contrôlent les actions des sprites.

Bloc Action
avancer de 10 pas Déplace le sprite de 10 pas
tourner de 90 degrés Fait pivoter le sprite
aller à x:100 y:50 Positionne le sprite aux coordonnées (100,50)

Exercice pratique

Créez un programme où un sprite suit un chemin triangulaire en utilisant :

  • Instructions de mouvement
  • Instructions de rotation
  • Bloc "répéter"

1.3 Les boucles

Définition : Une boucle permet de répéter une série d'instructions.

Bloc Scratch : répéter (10) fois

Exemple pour dessiner un cercle :

répéter (24) fois
    avancer de 10 pas
    tourner de 15 degrés
fin

Exercice pratique

Créez un programme qui dessine :

  1. Un hexagone régulier
  2. Une étoile à 5 branches
  3. Une spirale carrée

Chapitre 2 : Introduction à l’algorithme

Dans ce chapitre, nous avons exploré les concepts de base des algorithmes, y compris les variables, les instructions, les opérateurs et les organigrammes. Ces concepts sont essentiels pour résoudre des problèmes de manière structurée et logique. Les exercices pratiques permettent de renforcer ces compétences et de préparer les élèves pour le chapitre suivant sur l’intelligence artificielle.

2.1 Structure générale d’un algorithme

Définition : Un algorithme est une suite d’instructions logiques pour résoudre un problème. Il peut être représenté sous forme de texte, de code ou d’organigramme.

Exemple d’algorithme pour calculer la somme de deux nombres :

1. Demander à l’utilisateur de saisir le premier nombre.
2. Demander à l’utilisateur de saisir le deuxième nombre.
3. Ajouter les deux nombres.
4. Afficher le résultat.

Figure 2.1 : Organigramme d’un algorithme simple

Exercice pratique

Écrivez un algorithme pour calculer la moyenne de trois nombres.

2.2 Variables et constantes

Définition : Les variables stockent des données qui peuvent changer pendant l’exécution du programme, tandis que les constantes stockent des valeurs fixes.

Exemple :

  • Une variable peut stocker l’âge d’un utilisateur.
  • Une constante peut stocker la valeur de π (3.1416).
Élément Description
Variable Stocke des données modifiables (exemple : âge).
Constante Stocke des données fixes (exemple : π).

Exercice pratique

Créez un algorithme qui utilise une variable pour stocker le rayon d’un cercle et une constante pour π, puis calcule et affiche la circonférence.

2.3 Instructions

Définition : Les instructions sont des étapes spécifiques dans un algorithme qui décrivent une action à exécuter.

Exemple : Une instruction peut être "ajouter deux nombres" ou "afficher un message".

Exercice pratique

Écrivez un algorithme qui demande à l’utilisateur de saisir son nom, puis affiche un message de bienvenue.

2.4 Opérateurs

Définition : Les opérateurs permettent de manipuler des valeurs (addition, soustraction, multiplication, division, etc.).

Exemple : Utiliser l’opérateur + pour additionner deux nombres.

Opérateur Description
+ Addition
− Soustraction
× Multiplication
÷ Division

Exercice pratique

Écrivez un algorithme qui utilise des opérateurs pour calculer l’aire d’un rectangle.

2.5 Organigramme

Définition : Un organigramme est une représentation visuelle d’un algorithme à l’aide de symboles (rectangles, losanges, flèches, etc.).

Exercice pratique

Dessinez un organigramme pour un algorithme qui détermine si un nombre est positif, négatif ou nul.

2.6 Application avec Scratch et approche mathématique

Définition : Les algorithmes peuvent être appliqués dans Scratch pour résoudre des problèmes mathématiques, comme le calcul de l’aire ou du périmètre.

Exemple : Créer un programme Scratch pour calculer l’aire d’un triangle.

Exercice pratique

Créez un programme Scratch pour calculer le volume d’une sphère en utilisant la formule V = (4/3) × π × r³.

Chapitre 3 : Introduction à l’intelligence artificielle

Dans ce chapitre, nous explorons les concepts de base de l’intelligence artificielle (IA), en abordant des sujets comme l’éthique, la formulation de prompts efficaces et la réalisation de mini-projets. Ces notions permettent de comprendre le fonctionnement de l’IA et son utilisation responsable. Les exercices pratiques et débats en classe renforcent ces compétences et préparent à des sujets plus avancés en informatique.

3.1 Définition de l’intelligence artificielle

Définition : L’intelligence artificielle (IA) est la capacité d’une machine à imiter l’intelligence humaine, en apprenant, en raisonnant et en s’adaptant à de nouvelles situations.

Exemple : Les assistants virtuels comme Siri ou Alexa utilisent l’IA pour comprendre et répondre aux questions des utilisateurs.

Figure 3.1 : Représentation simplifiée de l’intelligence artificielle

Exercice pratique

Discutez en classe des exemples d’IA que vous utilisez dans votre vie quotidienne.

3.2 Éthique de l’intelligence artificielle

Définition : L’éthique de l’IA traite des questions morales liées à l’utilisation de l’intelligence artificielle, comme la protection des données, les biais algorithmiques et l’impact sur l’emploi.

Exemple : Les biais algorithmiques peuvent conduire à des décisions injustes, comme dans les systèmes de recrutement automatisés.

Problème éthique Description
Protection des données Comment les données personnelles sont-elles collectées et utilisées ?
Biais algorithmiques Les algorithmes peuvent-ils être injustes ou discriminatoires ?
Impact sur l’emploi L’IA remplace-t-elle des emplois humains ?

Exercice pratique

Organisez un débat en classe sur les avantages et les inconvénients de l’IA dans la société.

3.3 Prompt efficace

Définition : Un prompt est une instruction donnée à un système d’IA pour obtenir une réponse spécifique. Un prompt efficace est clair, précis et bien structuré.

Exemple : Au lieu de dire « Parle-moi de l’histoire », un prompt efficace serait « Explique-moi les causes de la Révolution française en 1789 ».

Exercice pratique

Écrivez trois prompts efficaces pour demander à une IA de résoudre des problèmes mathématiques.

3.4 Mini-projets

Définition : Les mini-projets permettent de mettre en pratique les concepts d’IA à travers des applications simples et concrètes.

Exemple : Créer un programme Scratch où un sprite reconnaît des formes géométriques.

Projet Description
Reconnaissance de formes Utiliser des règles simples pour identifier des formes géométriques.
Chatbot simple Créer un chatbot qui répond à des questions basiques.
Classification d’images Classer des images en fonction de leur contenu.

Exercice pratique

Créez un programme Scratch où un sprite utilise des règles simples pour résoudre des problèmes mathématiques.
